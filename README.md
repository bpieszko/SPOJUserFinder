# SPOJUserFinder
Script to look for people on Polish SPOJ Ranking.

## Making
```
g++-7 --std=c++17 -o SPOJUserFinder SPOJUserFinder.cpp -lcurl
```

## Using
```
./SPOJUserFinder "Bartosz Pieszko"
```

## Sample result
```
Found Bartosz Pieszko on position 50
```
